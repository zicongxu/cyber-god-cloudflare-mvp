# Cyber God Cloudflare MVP

赛博上帝 MVP 仓库，采用前后端分离结构。

## 目录

```text
frontend/      前端工程占位，交给前端同学实现
backend/       Cloudflare Workers 后端
docs/          前后端交互协议与开发约定
```

## MVP 闭环

```text
用户忏悔
→ 行为诊断
→ 生成审判
→ 派发救赎任务
→ 开启见证仪式
→ 用户确认完成 / 未完成
→ 奖励结算
→ 可选神谕
```

## 后端技术栈

```text
Cloudflare Workers
Cloudflare D1
TypeScript
```
